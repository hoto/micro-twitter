# Micro Twitter

A console-based micro scale twitter in Java.

### Clean

    ./gradlew clean
    
### Compile and test

    ./gradlew build

### Test

    ./gradlew test
    
### Build jar

    ./gradlew jar

### Run

    ./gradlew run
    
or

    ./gradlew jar
    java -classpath ./build/libs/micro-twitter.jar com.microtwitter.App
